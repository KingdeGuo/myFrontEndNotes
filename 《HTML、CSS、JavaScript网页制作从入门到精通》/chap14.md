# JavaScript脚本基础

- `onclick`触发点击事件。

- `onchange`触发改动事件。

- `onselect`是文本框中的内容被选中时所发生的事件。

- `onfocus`是光标在文本框上时产生的事件。

- `onload`是在加载网页时产生的事件。

- `onunload`是退出网页时引发是事件，并可更新cookie的状态。

- `onblur`是失去焦点时所触发的事件。

- `onmouseover`鼠标移动到对象范围的上方时触发的事件。

- `onmouseout`鼠标离开时触发的事件。

- `ondbclick`是鼠标双击时触发的事件。

- 浏览器内部对象

  - `navigator`是浏览器对象
  - `document`文档元素
  - `windows`处于对象层次最顶层。
  - `location`对象提供了与当前打开的URL一起工作的方法喝属性
  - `history`提供了历史清单有关的信息。
